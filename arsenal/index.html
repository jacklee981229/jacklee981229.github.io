    File Transfer

    These tools allow you to easily transfer files between your computer and mobile device.
    • LocalSend: Transfer files between your computer and mobile device.
    • Aconvert: Convert and edit files types, including PDFs.
    • File Converter: Convert files to different formats.

    Office Software

    These tools help you create, edit, and manage documents and other files.
    • Microsoft Office: Word, Excel, and PowerPoint.
    • Free KM Server: Activate Microsoft Office through a KMS server.
    • LibreOffice: A free and open-source alternative to Microsoft Office.

    Chrome Extensions

    These tools enhance your browsing experience by adding features and functionality to Google Chrome.
    • Extension Manager: Manage and optimize your Chrome extensions.
    • eJOY English: Learn English through movies with multi-language subtitles.
    • Todoist: A task management tool for Chrome.
    • Toby: An efficient tab management tool for Chrome.
    • Save to Pocket: Save web pages for offline reading.
    • Imagus: Preview images by hovering over them.
    • Full Page Screen Capture: Capture long web pages as images.
    • SuperCopy: To enable copying of text from websites that disable copy/paste.
    • Batch image downloader: To download multiple images at once.
    • ChatGPT Writer: To generates natural language text based on prompts.
    • ChatGPT for Google: To allows users to chat with ChatGPT directly from Google search results.
    • YouTube Summary with ChatGPT: To generates a summary of YouTube videos using ChatGPT.
    • WebChatGPT: To allows users to chat with ChatGPT from any website.
    • Luna: To generates natural language text based on user inputs.
    • ChatGPT Prompt Genius: To generates prompts for ChatGPT based on specific topics.

    Learning and Education

    Tools to facilitate online learning and improve educational experience.
    • My English Pages: Free online English learning.

    Entertainment and Creativity

    Tools to boost creativity and provide entertainment.
    • ToonMe: Face cartoon character generator.
    • Sharkle: Stimulating blind box website.
    • This Person Does Not Exist: AI-generated face photos.

    Cybersecurity and Network Protection

    Tools to ensure security and protect against cyber threats.
    • Cyberthreat Real-Time Map: Global network attack tracking.

    Multimedia

    Tools for multimedia content creation and management.
    • Handbreak: Video transcoder.
    • OBS: Free and open source software for video recording and live streaming.

    Productivity and Efficiency

    Tools to enhance productivity and efficiency.
    • Everything: File search utility.
    • Geekuninstaller: Efficient software uninstaller.
    • Listary: File search tool.
    • FolderMove: Move files from Disk C to D.

    Image and Video Processing

    Tools for image and video editing and processing.
    • Squoosh.app: Visual image online compression website.
    • 4k video downloader: Download video on all links.
    • Sandboxie: Sandboxing tool.
    • F.lux F: Eye protection program.

    Miscellaneous Tools

    Tools that don’t fit into other categories or serve multiple purposes.
    • Brackets: A lightweight document viewer.
    • SubHD: Chinese subtitles for movies and TV shows.
    • Ninite: A custom package of common useful apps.
    • Umi-OCR: Converts scanned PDFs/images to editable text.
    • Wolfram Alpha: Computational knowledge engine for math and facts.
    • XAMPP: Cross-platform web server with Apache, MySQL, PHP, and Perl.
    • YouTube-dl: Command-line program to download videos from sites.
    • Zenkit: Tool for managing projects and tasks.
    • MuscleWiki: Fitness information database with articles and workout plans.
    • MagicSlides: To generates slides based on user inputs using ChatGPT.
    • TreeSize Free: Disk space analyzer.
    • AI online drawing + prompts: AI painting keyword website recommendation.
    • Pico: A tiny, hackable BitTorrent client.
    • Android phone screen mirroring: Display and control your Android device.
